## The application
The AçaiGo App gives you the quickest, easiest way to place an order for acai bowls and then go to the store to pick them up.

### Features
-----

* Login
* Registration
* See product details
* Order a product
* Edit the quantity of the product
* See the user profil
* Edit the user email
* Delete the user account
